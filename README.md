# CamScanner

### You can run the application
#### Step : 1
##### Clone the Github repo into your system.
#### Step : 2
##### Open the folder in any code editor(vscode) and type :- npm install  [to upload all the packages]
#### Step : 3
##### run the command :- npx react-native run-android
